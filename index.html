新古日本

作家：Rizki Kadir ウェブペジ：http://rkadir-nihon.paperplane.io/

日本を楽しく見たり、生活と文化を習ったりしましょう！

この旅は一般的な旅じゃありません。この旅は一週間日本の歴史と近代的な生活を共に習える旅です。最初は東京に到着してから、すぐ新幹線で京都に行きます。新幹線から広い日本の地方が見られます。京都は明治時代まで日本の首都でした。そのため歴史的な建物が多いです。次、京都から天橋立に出発します。天橋立には古い所が多くてきれいな景色もあります。それで、東京と大阪には近代的な日本の生活を経験することができます。帰る前に、東京の銀座で買い物もできますよ。私たちの準備した旅程をしてから、たしかに気持ちよくなります。

ですから、待たないで、今すぐ私たちの旅を予約してください。

どこえいきますか

  1. （月曜日）成田空港 -> 東京 -> 京都

  2. （火曜日）京都 -> 福知山 -> 天橋立

  3. （水曜日）天橋立の見物

  4. （木曜日）天橋立 -> 大阪

  5. （金曜日）東京 -> 成田空港

天橋立の観光

天橋立には色々な観光地があるけれども、まず天橋立ビュランドへ行きます。天橋立ビュランドは高い山にあるので、きれいな景色が見られます。その景色は飛龍観（ひりゅうかん）という景色です。飛んでいる竜（りゅう）ような地形だから、その名前を受けました。確かに天橋立で過ごして経験は忘れられる経験です。

東京の銀座

どうしてそんな所へいきたいんですか。銀座は買い物の所だけではなく、面白くて歴史的な所です。この場所から日本の近代的な生活と日常が見られます。散歩しながら、どう明治時代から今まで変わっていますかと聞けます。銀座で忙しくてにぎやかな道を見回してから、すばらしい浜離宮恩賜庭園（はまりきゅうおんしていえん）へ行きます。この庭園は中央の東京にあるのに、きれいな植物と花が見られます。

値段と宿泊

心配なくて、日本に到着から帰るまで全部整理します。私達の予約している宿泊は主に大きいな駅の近くてとても便利です。電車の切符を買うも整えます。それに、新幹線で使うので、交通の時間は長くないです。ですから、この旅を簡単に楽しむ。
